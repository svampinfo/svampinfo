# Svampinfo

[Svampinfo.se](https://svampinfo.se) är en sida för dig som älskar svampar, och
för dig som är intresserad av att lära dig mer

## Bidra

Du kan bidra till sidan på massvis med vis. Antingen genom att berätta om sidan
för vänner, eller genom att bidra mer direkt genom att skriva bättre information
om svamparna, lägga till bilder osv. Om du vill lägga till "innehåll" är det enklast
om du gör följande:

1. Öppna en ["issue"](https://github.com/svampinfo/svampinfo/issues) angående
   vad du vill lägga, ändra eller vad någon annan bör göra.
2. Efter att du har fått OK i _issuet_, kan du själv välja att implementera det.
3. Skapa en ["fork"](https://github.com/svampinfo/svampinfo/fork) av repot.
4. Ändra det du vill ändra, och skapa en commit.
5. Starta en ["Pull request"](https://github.com/svampinfo/svampinfo/compare),
   vilket är en förfrågan att dra in dina ändringar.

När du bidrar med kod eller med innehåll går du med på att tillgängliggöra det
i ovanstående licenser.
